# CherryChat
## About
A very simple WebUI for ollama, currently only supports sending/recieving messages and attaching images (automatically scaled to 520px). No configuration UI has been implemented yet and the model is set to `qwen3.5:9b`.
## How to run
This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.3. You can run it using the following commands:
1. `npm i` to install required packages
2. `ng serve` to host the UI locally
